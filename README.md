# 📊 Gerador de Relatórios em PDF

Este projeto é uma aplicação web interativa que permite gerar relatórios personalizados com gráficos, tabelas e análises, e exportá-los em formato PDF com alta qualidade. Criado com **HTML**, **Bootstrap 5**, **Tailwind CSS**, **Chart.js**, **jsPDF** e **html2canvas**.

---

## 🚀 Funcionalidades

- 🎯 Título e período personalizável do relatório
- 📊 Gráficos interativos e configuráveis (linhas, barras, pizza)
- 🎨 Esquema de cores customizável (padrão, vibrante, pastel)
- 📈 Dados dinâmicos de vendas, clientes e receita
- 📋 Tabela de dados automatizada
- 📝 Campo de análise descritiva do relatório
- 👁️ Visualização em tempo real (modo preview)
- 📤 Exportação para **PDF com layout estruturado**
- 📱 Interface responsiva e amigável

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia       | Finalidade                          |
|------------------|--------------------------------------|
| HTML + CSS       | Estrutura e layout geral             |
| Bootstrap 5      | Componentes responsivos              |
| Tailwind CSS     | Estilização rápida e moderna         |
| Chart.js         | Gráficos interativos                 |
| jsPDF + autoTable| Geração e exportação de PDF          |
| html2canvas      | Captura de gráficos como imagem      |
| Font Awesome     | Ícones ilustrativos                  |

---

## 📦 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GbrlSouza/gerador-relatorios-pdf.git
   ```

2. Abra o arquivo `index.html` no seu navegador:

   * Não precisa de servidor, funciona totalmente localmente.

3. Personalize:

   * Título, tipo de gráfico, período, esquema de cores, análise e pronto!

4. Clique em:

   * `Visualizar Relatório` para abrir o preview.
   * `Exportar para PDF` para gerar e baixar o arquivo.
